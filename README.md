# 🏰 PixelDungeon Dash: D&D Edition

Высокодетализированный бесконечный раннер в стиле темного фэнтези, созданный специально для Telegram WebApp. Исследуйте опасные глубины подземелья, сражайтесь с монстрами и соревнуйтесь в глобальном Зале Славы!

## ✨ Особенности игры

- **Система классов D&D**: Выберите одного из 4 уникальных героев с индивидуальными характеристиками и пассивными способностями.
- **Глобальная Таблица Рекордов (Hall of Fame)**: Соревнуйтесь в реальном времени. Система отслеживает Топ-10 игроков и ваше персональное место в общем зачете.
- **Атмосферный Pixel-Art**: Динамический параллакс-фон с перекрывающимися колоннами, факелами и частицами пыли для создания глубокой атмосферы подземелья.
- **D&D Механика**: Расчет урона через проверку Класса Брони (AC), критические успехи и провалы d20.
- **Полноэкранная Адаптивность**: Игра автоматически подстраивается под любое разрешение экрана в Telegram, обеспечивая бесшовный игровой опыт без черных полос.

## 🛡️ Герои и Способности

- **Воин (Fighter)**: Настоящий танк. Самая высокая броня (AC 15) и 6 HP. Идеально для тех, кто предпочитает блокировать удары.
- **Плут (Rogue)**: Мастер маневров. Обладает врожденным двойным прыжком для преодоления самых сложных препятствий.
- **Маг (Wizard)**: Магический поиск. Получает на 25% больше очков опыта, но имеет всего 3 HP.
- **Бард (Bard)**: Песнь отдыха. Автоматически восстанавливает 1 HP каждые 20 секунд.

## 👾 Бестиарий (Детализированные монстры)

- **Древний Дракон**: Огромный противник с анимированными перепончатыми крыльями и многослойным извергающимся пламенем.
- **Мимик**: Жуткий сундук, который клацает зубами и следит за вами множеством глаз, прячущихся в темной щели крышки.
- **Огр**: Массивный враг с огромной шипованной дубиной, совершающей круговые удары.
- **Бехолдер, Слизь, Призраки и Летучие мыши**: Уникальные паттерны движения и детальная прорисовка хитбоксов для честной игры.

## 🚀 Технологический стек

- **Frontend**: Next.js 15 (App Router), React, Tailwind CSS.
- **Backend/DB**: Firebase Firestore (хранение рекордов и данных окружения).
- **Auth**: Firebase Authentication (Анонимные сессии для мгновенного входа без лишних регистраций).
- **UI Components**: Shadcn UI & Lucide Icons.
- **Game Engine**: Custom HTML5 Canvas Game Loop с оптимизированной системой коллизий (AABB).

## 🛠️ Разработка и Архитектура

Игра использует архитектуру "клиентского управления данными" с интеграцией Firebase. Все игровые ассеты отрисовываются программно на Canvas, что обеспечивает 60 FPS даже на слабых мобильных устройствах. Система безопасности Firestore гарантирует, что игроки могут сохранять только свои собственные рекорды.
