# 🏰 PixelDungeon Dash: D&D Edition

Высокодетализированный бесконечный раннер в стиле темного фэнтези, созданный специально для Telegram WebApp. Исследуйте опасные глубины подземелья, сражайтесь с монстрами и соревнуйтесь в глобальном Зале Славы!

## ✨ Особенности игры

- **Система классов D&D**: Выберите одного из 4 уникальных героев с пассивными способностями.
- **Глобальная Таблица Рекордов**: Соревнуйтесь с другими игроками в реальном времени. Система отслеживает ваш личный ранг в общем зачете.
- **Атмосферный Pixel-Art**: Многослойный параллакс-фон с динамическим освещением, частицами пыли и проработанными монстрами.
- **D&D Механика**: Расчет урона через проверку Класса Брони (AC), критические успехи и провалы.
- **Адаптивность**: Игра идеально работает на любых мобильных устройствах и внутри Telegram.

## 🛡️ Герои и Способности

- **Воин (Fighter)**: Настоящий танк. Самая высокая броня (AC 15) и 6 HP.
- **Плут (Rogue)**: Мастер маневров. Обладает врожденным двойным прыжком.
- **Маг (Wizard)**: Магический поиск. Получает на 25% больше очков опыта, но имеет всего 3 HP.
- **Бард (Bard)**: Песнь отдыха. Автоматически восстанавливает 1 HP каждые 20 секунд.

## 👾 Бестиарий

- **Дракон**: Изрыгает пламя и бьет мощными крыльями.
- **Мимик**: Жуткий сундук, который клацает зубами и следит за вами множеством глаз.
- **Огр**: Огромный противник с вращающейся шипованной дубиной.
- **Слизь, Призраки и Летучие мыши**: Классические обитатели глубин с уникальными паттернами движения.

## 🚀 Технологический стек

- **Frontend**: Next.js 15 (App Router), React, Tailwind CSS.
- **Backend/DB**: Firebase Firestore (Real-time данные).
- **Auth**: Firebase Authentication (Анонимные сессии для мгновенного входа).
- **UI Components**: Shadcn UI & Lucide Icons.
- **Game Engine**: Custom Canvas Game Loop с использованием `requestAnimationFrame`.

## 🛠️ Разработка

Игра построена на архитектуре "клиентского управления данными" с использованием Firebase Studio. Все игровые ассеты отрисовываются программно на HTML5 Canvas, что обеспечивает высокую производительность и плавность анимаций.
